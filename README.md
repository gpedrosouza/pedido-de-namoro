# Pedido de Namoro

Landing page romântica em HTML, CSS e JavaScript, preparada para apresentar fotos, textos e uma linha do tempo de momentos especiais.

## Estrutura

- `index.html`: marcação principal da página
- `style.css`: estilos e responsividade
- `script.js`: interações e animações
- `images/`: imagens usadas na experiência

## Como abrir

Basta abrir o arquivo `index.html` no navegador.

## Personalização

- Substitua as imagens dentro de `images/`
- Ajuste os textos diretamente em `index.html`
- Refine cores, espaçamentos e animações em `style.css`
